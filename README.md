# Case-Social-Connect
Repositório que servirá como 1º Checkpoint da disciplina de WebDev da FIAP

SocialConnect
    
Resumo do Projeto:

    O SocialConnect é um site institucional desenvolvido para apresentar uma empresa fictícia do setor automotivo especializada em carros elétricos de alta performance.

    O objetivo do projeto é demonstrar informações sobre a empresa, como sua proposta, equipe e formas de contato, além de oferecer uma navegação simples e organizada para o usuário. O site foi estruturado com foco em clareza, organização das seções e visual moderno.

A página possui:

    Cabeçalho com navegação
    Seção "Sobre nós"
    Seção da equipe
    Informações de contato
    Rodapé com redes sociais

Tecnologias Utilizadas:

    HTML5 – Utilizado para estruturar o conteúdo da página.
    CSS3 – Utilizado para estilização, layout e organização visual do site.
